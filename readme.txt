[tsnd151.pde]
１．pcと加速度センサを接続後、[42行目]のシリアルポートを変更してください。
２．プログラムを起動させる際には、センサを平らな場所に置いてください。
３．重力軸の回転については、"プログラム起動時"の加速度センサの向きが初期値となります。
　　加速度センサに記載されているy軸を正面に向けてからプログラムを起動すると、加速度センサと表示の動作が一致します。

４．角度データは3つまで保存できます。
　　数字キー1,2,3を押した瞬間から、保存を開始します。
　　押した数字キーにより、保存先が変わります。(数字キー1~3 ⇒ ./csv/degs1~3.csv)
　　保存先にデータが存在するとき、上書き保存されます。
　　データの保存周期は29行目bpmから変更が可能ですが、設定によっては誤作動が生じる可能性がございます。(初期設定：10Hz)

５．Qキーを押すとcsvファイルにデータが書き込まれ、プログラムが終了します。
　　Qキーを押さずにウィンドウを閉じるとデータは保存されませんので、ご注意ください。
　　また、保存を開始して5分後に、強制的にデータが保存され、プログラムが終了します。
　　保存時間上限の変更は30行目time_maxから行えます。(初期設定：300s)

[tsnd151_replay.pde]
０．数字キー1,2,3で保存した角度データを再生できます。
１．Eキーでリプレイの中止を行えます。
２．Qキーでプログラムを終了します。
